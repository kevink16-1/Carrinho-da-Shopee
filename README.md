# Carrinho-da-Shopee
Projeto desenvolvido no bootcamp da DIO que simula um carrinho da Shopee em JavaScript. Permite adicionar, remover e atualizar itens, calcular subtotal e total e visualizar o resumo da compra de forma clara. É uma prática direta de lógica, funções e organização de código em um cenário real de e-commerce. com foco em prática e entendimento do fluxo.
